Resumo das apresentações de Trabalho de Conclusão de Curso (TCC) relacionadas à disciplina de programação web:

TCC: "Desenvolvimento de um Sistema de Comércio Eletrônico para Pequenas Empresas"

Objetivo: O trabalho buscou desenvolver um sistema de comércio eletrônico personalizado para pequenas empresas, visando facilitar suas operações de vendas online.
Programação web: Foram utilizadas tecnologias como HTML, CSS e JavaScript para a criação da interface do sistema. A parte de backend foi desenvolvida utilizando PHP e banco de dados MySQL.
Contribuição: O TCC apresentou uma abordagem prática para o desenvolvimento de sistemas de comércio eletrônico personalizados, fornecendo um exemplo de implementação utilizando tecnologias comumente empregadas na programação web.
Avaliação crítica: As técnicas de programação web utilizadas foram eficazes na criação do sistema, e a apresentação foi clara. No entanto, seria interessante explorar mais a segurança do sistema e discutir opções de integração com plataformas de pagamento.

TCC: "Otimização de Desempenho em Aplicações Web Responsivas"

Objetivo: O trabalho teve como objetivo otimizar o desempenho de aplicações web responsivas, a fim de proporcionar uma melhor experiência de usuário em dispositivos móveis.
Programação web: O TCC abordou técnicas de otimização, como compressão de recursos, cache de conteúdo e carregamento assíncrono, utilizando linguagens e frameworks web, como HTML, CSS, JavaScript e jQuery.
Contribuição: O trabalho apresentou práticas recomendadas para melhorar o desempenho de aplicações web responsivas, contribuindo para a melhoria da eficiência e velocidade das aplicações desenvolvidas nesse contexto.
Avaliação crítica: As técnicas de otimização propostas foram bem aplicadas e a apresentação foi clara e relevante para o campo da programação web. No entanto, seria interessante abordar também o impacto dessas otimizações na compatibilidade com diferentes navegadores e dispositivos.

TCC: "Aplicação de Progressive Web Apps para Melhorar a Experiência do Usuário em Sites de Notícias"

Objetivo: O trabalho teve como objetivo aplicar Progressive Web Apps (PWAs) em sites de notícias, visando melhorar a experiência do usuário, mesmo em condições de conectividade instável.
Programação web: O TCC explorou as tecnologias e técnicas de programação web utilizadas na criação de PWAs, como Service Workers, Web App Manifest e IndexedDB. Também foram utilizados HTML, CSS e JavaScript para o desenvolvimento da interface.
Contribuição: O TCC apresentou uma abordagem inovadora para melhorar a experiência do usuário em sites de notícias, utilizando PWAs, que oferecem recursos como carregamento rápido, notificações push e capacidade de funcionar offline.
Avaliação crítica: A aplicação das técnicas de programação web relacionadas a PWAs foi eficaz e a apresentação foi clara. No entanto, seria interessante discutir os desafios de implementação e a compatibilidade com diferentes navegadores e dispositivos.

Conclusão geral:
Os TCCs apresentados contribuíram significativamente para o campo da programação web, abordando temas relevantes e apresentando soluções práticas. Eles exploraram tecnologias, linguagens, frameworks e técnicas específicas de programação web, demonstrando sua aplicabilidade e impacto na criação de sistemas de comércio eletrônico, otimização de desempenho em aplicações responsivas e aprimoramento da experiência do usuário em sites de notícias.

No entanto, é importante ressaltar que essas contribuições podem ser ainda mais aprimoradas. Aspectos como segurança, compatibilidade com diferentes navegadores e dispositivos, integração com outras tecnologias e a consideração de boas práticas em design e usabilidade são áreas que podem ser exploradas para um aprofundamento futuro.

Em geral, os TCCs evidenciaram a importância da programação web no desenvolvimento de soluções práticas e eficazes, fornecendo exemplos concretos de como as tecnologias e técnicas específicas podem ser aplicadas para resolver problemas reais e melhorar a experiência do usuário.
